Developer Edit Mode

If you started a cleric or magic user before the spell components were installed click this button to update your character raw Pressing this button will erase your spellbook and spells if you already have them.

While this project is under active development and some key features are missing from the proper UI this view will exist to let you tweak and update your character by editing the raw JSON

You can mess up your character and cause unusual behavior. Please save your character before making changes
